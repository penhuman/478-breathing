4-7-8 呼吸放鬆引導 (4-7-8 Breathing Exercise)

這是一個簡單且優雅的網頁應用程式，旨在透過視覺引導協助使用者進行 4-7-8 呼吸法 練習，以達到放鬆神經、減輕焦慮並幫助入睡的效果。

🔗 線上體驗： [在此貼上您的 GitHub Pages 網址]

✨ 特色

直觀的視覺引導：

🔵 吸氣 (4秒)：圓圈變大，發出藍光。

⚪ 憋氣 (7秒)：圓圈保持最大，發出白光並帶有微幅波動。

⚫ 吐氣 (8秒)：圓圈縮小，回歸平靜暗色。

現代化 UI 設計：採用 Tailwind CSS 打造深色動態漸層背景與玻璃擬態 (Glassmorphism) 介面，保護眼睛並提供沉浸式體驗。

完全響應式：完美支援手機、平板與電腦瀏覽器。

隱私安全：純前端程式 (HTML/JS)，所有運作皆在您的瀏覽器中執行，不收集任何數據。

🚀 如何使用

開啟網頁。

跟隨畫面中央的文字指示調整呼吸。

點擊「開始練習」按鈕啟動引導。

隨時可點擊「停止」或重置練習。

🛠️ 技術棧

HTML5

JavaScript (ES6+): 使用 Async/Await 與 Promise 處理精確的計時動畫。

Tailwind CSS (CDN): 快速構建樣式與動畫效果。
